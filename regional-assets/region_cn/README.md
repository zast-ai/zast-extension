# Zast.ai 安全评估扩展

一个与 Zast.ai 平台集成的 VSCode 扩展，为 Java 应用程序提供自动化安全漏洞评估。

## 功能特性

- **OAuth 认证**: 安全登录到 Zast.ai 平台
- **安全评估**: 创建和管理自动化安全评估任务
- **Java 项目支持**: 自动发现 Maven 和 Gradle 项目构件
- **隧道管理**: 内置 Cloudflare 隧道集成，用于暴露本地应用程序
- **报告查看**: 直接在 VSCode 中查看安全报告和执行 POC 脚本
- **任务管理**: 跟踪评估进度和管理多个安全任务

## 系统要求

- VSCode 1.85.0 或更高版本
- Node.js 16.x 或更高版本
- 使用 Maven 或 Gradle 构建系统的 Java 项目
- 有效的 Zast.ai 账户

## 安装说明

### 从 VSCode 扩展市场安装

1. 打开 VSCode
2. 转到扩展 (Ctrl+Shift+X)
3. 搜索 "Zast.ai Security Assessment"
4. 点击 "安装"

### 从 VSIX 包安装

1. 从发布页面下载 `.vsix` 文件
2. 打开 VSCode
3. 转到扩展 (Ctrl+Shift+X)
4. 点击 '...' 菜单并选择 '从 VSIX 安装...'
5. 选择下载的 `.vsix` 文件

或使用命令行:

```bash
code --install-extension zast-extension-0.0.6.vsix
```

## 快速开始

1. **登录 Zast.ai**

   - 从活动栏打开 Zast.ai 面板
   - 在首页视图中点击 "登录"
   - 在浏览器中完成 OAuth 认证流程

2. **开始安全评估**

   - 在首页视图中点击 "开始新评估"
   - 扩展将自动发现您的 Java 项目构件
   - 配置测试环境服务 URL
   - 设置用于认证测试的用户会话
   - 审查并开始评估

3. **查看报告**
   - 评估结果将显示在 "项目任务" 视图中
   - 点击任何任务以查看详细的安全报告

## 扩展视图

扩展在 Zast.ai 面板中提供三个主要视图:

- **首页**: 主仪表板，显示认证状态和快速操作
- **项目任务**: 当前项目的安全评估任务
- **所有任务**: 所有安全评估任务的完整列表

## 安全注意事项

**重要提示**: 此扩展可能会通过 Cloudflare 隧道将您的本地应用程序暴露到互联网。请确保:

- 您的应用程序不包含敏感的生产数据
- 为您的测试环境配置了适当的认证
- 您了解公开暴露的安全影响
- 不再需要时停止隧道
